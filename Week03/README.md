## 下载
```
go get -u golang.org/x/sync.
``` 
下载不成功，考虑git clone 到相应文件目录中
```
git clone https://github.com/golang/sync.git
```

You can also manually git clone the repository to $GOPATH/src/golang.org/x/sync.
